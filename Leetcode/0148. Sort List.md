`Middle`

> 在 $O(n\log n)$ 时间复杂度和**$O(1)$空间复杂度**下，对链表进行排序。

#### 1. 归并排序（递归）

- cut
    - 快慢指针找中点，从中点断开链表
- merge：合并两个有序链表  [0021. Merge Two Sorted Lists.md](0021. Merge Two Sorted Lists.md)  
    - 时间复杂度 $O(n+m)$

```python
class Solution:
    def sortList(self, head: ListNode) -> ListNode:
        if not head or not head.next: return head # termination
        # cut the LinkedList at the mid index
        slow, fast = head, head.next
        while fast and fast.next:
            slow, fast = slow.next, fast.next.next
        mid, slow.next = slow.next, None # save and cut  mid:奇数个节点则为中点，否则为中心右边的节点
        # recursive for cutting.
        left, right = self.sortList(head), self.sortList(mid)
        
        # merge `left` and `right` linked list and return it
        h = ans = ListNode(0)
        while left and right:
            if left.val < right.val: h.next, left = left, left.next
            else: h.next, right = right, right.next
            h = h.next
        h.next = left if left else right
        return ans.next
```

:warning: 空间复杂度： $O(\log n)$, 即递归深度

#### :+1: 2. 归并排序（直接自底向上两两合并有序链表）

```python
class Solution:
    def sortList(self, head: ListNode) -> ListNode:
        h, length, intv = head, 0, 1
        while h: h, length = h.next, length + 1   # count the length of the linkedlist
        ans = ListNode(0)
        ans.next = head
        # merge the list in different intv.
        while intv < length:  # termination
            pre, h = ans, ans.next
            while h:
                # get the two merge head `h1`, `h2`
                h1, i = h, intv
                while i and h: h, i = h.next, i - 1
                if i: break # no need to merge because the `h2` is None.
                h2, i = h, intv
                while i and h: h, i = h.next, i - 1
                c1, c2 = intv, intv - i # the `c2`: length of `h2` can be small than the `intv`
                # merge the `h1` with length c1 and `h2` with length c2
                while c1 and c2:
                    if h1.val < h2.val: pre.next, h1, c1 = h1, h1.next, c1 - 1
                    else: pre.next, h2, c2 = h2, h2.next, c2 - 1
                    pre = pre.next
                pre.next = h1 if c1 else h2
                while c1 > 0 or c2 > 0: 
                    pre, c1, c2 = pre.next, c1 - 1, c2 - 1
                pre.next = h 
            intv *= 2
        return ans.next
```

空间复杂度： $O(1)$
