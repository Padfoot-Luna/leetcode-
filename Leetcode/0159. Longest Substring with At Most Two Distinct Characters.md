`Middle`

> 给定一个字符串 s ，找出 **至多** 包含两个不同字符的最长子串 t ，并返回该子串的长度。

#### 1.  滑动窗口+双指针

```python
class Solution:
    def lengthOfLongestSubstringTwoDistinct(self, s: str) -> int:
        ans = 0
        left = 0
        hashmap = {}
        for right in range(len(s)):
            if s[right] in hashmap:
                hashmap[s[right]] += 1
            else:
                hashmap[s[right]] = 1
            while len(hashmap) > 2:   # 删除前面的字母
                if hashmap[s[left]] > 1:
                    hashmap[s[left]] -= 1
                else:
                    del hashmap[s[left]]
                left += 1
            ans = max(ans, right-left+1)
        return ans
```
